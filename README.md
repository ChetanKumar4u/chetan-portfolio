Chetan Kumar - Portfolio Website
A modern, animated portfolio website built with React, Tailwind CSS, and Framer Motion. Designed to showcase projects, skills, and make a great impression on recruiters.

Features
Responsive Design - Works seamlessly on all devices
Smooth Animations - Fade-up animations on scroll using Framer Motion
Modern Dark Theme - Sleek dark background with cyan/purple accent colors
Glass Morphism UI - Modern frosted glass effects with backdrop blur
Project Showcase - Display your best projects with descriptions, tags, and links
Skills Section - Highlight technical expertise
Contact Form - Enable visitors to reach out
Social Links - GitHub and LinkedIn integration
Fast & Optimized - Built with Vite for quick performance

Tech Stack
React 18
Vite
Tailwind CSS
Framer Motion
React Icons

Project Sections
Hero - Eye-catching introduction with skills preview
About - Your background and interests
Skills - Technical skills and tools
Projects - Showcase of work with live demo and GitHub links
Contact - Contact form for inquiries

Getting Started
npm install
npm run dev      # Development server
npm run build    # Production build
